# Консольное приложение для управления задачами через CRUD-операции

Поддерживает команды:
- add
- list
- edit (указать через пробел после команды id задачи)
- delete (указать через пробел после команды id задачи)
- filter (указать через пробел после команды статус задачи (todo/in_progress/done)
- sort
- exit
